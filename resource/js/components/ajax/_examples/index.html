» QW.Ajax.get(url, [data], [callback])

GET方式的Ajax请求

参数：

url {String|HTMLFormElement}
Ajax请求的url。
data {String|Object|HTMLFormElement}
发送的数据。
callback(responseText) {Function}
请求完成后的回调函数，responseText {String}为服务器响应的字符串。

例子1：url参数为HTMLFormElement对象，data参数忽略

例子2：url参数为url（包含query string），data参数忽略

例子3：url参数为url，data参数为JSON对象

例子4：url参数为url，data参数为HTMLFormElement对象

QW.Ajax.post(url, [data], [callback])用法和QW.Ajax.post(url, [data], [callback])完全相同。只需要把示例中的get换成post即可。
